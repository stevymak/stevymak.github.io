Installation et réinstallation
Windows à domicile

Un PC qui rame depuis des mois, un système Windows corrompu après un virus, un nouveau disque dur à configurer ou simplement l'envie de repartir sur une base propre : la réinstallation de Windows est souvent la meilleure solution. Stevy intervient à domicile en Île-de-France pour effectuer cette opération dans les règles de l'art, sans que vous perdiez une seule donnée.

La prestation est complète : sauvegarde intégrale de vos fichiers avant toute opération, installation propre de Windows 10 ou 11, installation de tous les pilotes (drivers), configuration des logiciels essentiels (navigateur, antivirus, Office si besoin) et mise en place de la sécurité. Vous récupérez un PC comme neuf, avec toutes vos données.

Questions fréquentes

Mes données seront-elles effacées ? +
Stevy sauvegarde TOUT avant la réinstallation : photos, documents, contacts, favoris. Les données sont restituées après l'installation propre. La sauvegarde est vérifiée et confirmée avant de procéder à quoi que ce soit.
Quelle version de Windows est installée ? +
Windows 10 ou Windows 11 selon la compatibilité de votre PC. Si votre PC a moins de 4 Go de RAM ou un processeur trop ancien, Windows 10 reste recommandé pour de meilleures performances. Stevy vous conseille la version adaptée.
Combien de temps dure une réinstallation complète ? +
Environ 2h30 à 3h pour une installation complète avec logiciels. Il faut prévoir le forfait Long format (129€ / 64,50€ net*). Ce temps inclut la sauvegarde, l'installation et la configuration.
La clé Windows est-elle incluse ? +
Si votre PC est livré avec Windows (OEM — cas des PC du commerce), la clé est intégrée dans le BIOS et sera reconnue automatiquement. Sinon, Stevy vous indique comment l'obtenir légalement auprès de Microsoft, sans passer par des sources douteuses.

Besoin d'une réinstallation Windows ?

Repartez sur une base propre, sans perdre vos données. Intervention à domicile en Île-de-France.
